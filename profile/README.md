# LayerPay.org — offshore crypto payments for business
                                                                                                                                    
  LayerPay.org is a privacy-first payment gateway for businesses that need to accept crypto without KYC or complex compliance. Simple dashboard, 4 networks, lowest fees in the industry.                                                                                                                                              
   
  ## Why it matters                                                                                                                                                  
  - Accept TRX, ETH, BSC, SOL, USDT, USDC without KYC requirements.                  
  - Support for Ethereum, BSC, Tron and Solana networks.
  - Instant settlement with automatic payouts.
  - Lowest fees: $0.50 + 0.75% per transaction.
  - Full API for seamless integration.

  ## Getting started
  1. Visit `https://layerpay.org`.
  2. Connect your MetaMask or Phantom wallet.
  3. Create your first app and generate an API key.
  4. Enable the networks you need.
  5. Start accepting crypto payments in minutes.
