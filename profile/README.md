# LayerPay.fi — crypto payments for business

LayerPay.fi is a web interface (and an API) for teams that need a reliable way to accept crypto payments without complex setup. The project combines a flexible dashboard, integrations with 30+ networks, and a lot more.

## Why it matters
- Connect crypto payments to apps and marketplaces.
- Manage API keys and payment apps from one place.
- Enable or disable networks per business need.
- See payment analytics in real time.

## Getting started
1. Visit `https://layerpay.fi`.
2. Sign in using your preferred method (email, Google, MetaMask, TON).
3. Create your first app and generate an API key.
4. Enable the networks you need and set limits or rules.
5. Plug the key into your integration and track results in the dashboard.
